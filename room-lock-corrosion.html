Lock Corrosion

Use redox chemistry to weaken and corrode the lock mechanism without damaging nearby components.

In-Depth Room Explanation

This room tests oxidation-reduction logic. You must select reagents that transfer electrons effectively to corrode the lock at the right rate.

  • Match oxidizing and reducing agents correctly
  • Apply reaction in phases to avoid over-corrosion penalties
  • Observe lock state changes after each reaction step

Clear strategy: Prime the lock surface first, then run the main redox step. Trigger the unlock action immediately when corrosion reaches threshold.
